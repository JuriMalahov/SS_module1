1. Для работы код нужно установить необходимые библиотеки: pandas, numpy, natasha, nltk,
sklearn, pymorphy2;
2. Запустите код и введите в терминале название файла, с которым он будет работать.
3. Затем введите в терминале слово, словосочетания с которым вы хотите получить.
4. В результате вы получите три списка: "Существительное в прямом значении + прилагательное",
"Существительное + прилагательное в переносном значении" и
"Существительное в переносном значении + прилагательное".
